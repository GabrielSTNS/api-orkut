# 🚀 API de Autenticação de Usuários

API desenvolvida em Node.js para gerenciamento de usuários com autenticação JWT e criptografia de senhas com bcrypt

## 📌 Funcionalidades

- Cadastro de usuários
- Login com geração de token JWT
- Listagem de usuários (protegida)
- Cadastro de publicações
- Listagem de publicações
- Alteração de publicações
- Exclusão de publicações

## 💻 Tecnologias utilizadas

- Node.js
- Express.js
- PostgreSQL
- bcrypt
- JWT
- dotenv
- Joi

## ⚙️ Pré-requisitos

- Node.js instalado
- PostgreSQL instalado
- Gerenciador de pacotes (npm ou yarn)
